# Pentagram-Animation

A small, self-contained animation that draws and animates a pentagram using HTML, CSS and JavaScript.

![Demo Screenshot](./Screenshot%202024-12-08%20001455.png)

Live preview

- Open `index.html` in your browser (double-click the file) or serve the directory with a static server (for example `npx serve` or `live-server`).

Highlights

- Draws a pentagram and animates it using vanilla JavaScript and CSS.
- Minimal, dependency-free single-file demo (`index.html`).
- Easy to fork and experiment with the animation parameters.

Files

- `index.html` — the demo page with the animation.
- `Screenshot 2024-12-08 001455.png` — screenshot used above.

Usage

1. Clone or download this repository:

   git clone https://github.com/BinaryVortex/Pentagram-Animation.git

2. Open `index.html` in a browser or serve the folder:

   npx serve

3. Tweak the animation and styling inside `index.html` to experiment.
